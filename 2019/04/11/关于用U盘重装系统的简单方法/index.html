关于用U盘重装系统的简单方法

发表于 2019-04-11 | 分类于 学习笔记 | | 阅读次数: | 阅读数
字数统计: 299 | 阅读时长 ≈ 1

前言

  • 网上有很多用U盘重装系统的方法，都需要在U盘中创建pe启动，但是我最近发现，U盘重装系统根本不需要这样操作。

方法介绍：

  • 其实只需要将下载好的windows镜像，也就是ISO文件解压到U盘里就可以了，然后开机进入BIOS设置U盘启动就可以了！
1、下载官方的windows镜像文件：
  • 这里我推荐比较好的官方原镜像下载地址：http://msdn.itellyou.cn 找到想要安装的镜像文件迅雷下载就行了。
2、将U盘（大于等于8g）格式化为NTFS格式
3、把下载好的镜像文件用解压工具解压到U盘里
  • 直接解压到U盘根目录即可，不要创建任何文件夹！
4、备份好重要文件，关机重启，插上U盘，设置U盘启动，等待安装完成即可
  • U盘启动设置以及安装的细节步骤网上有很多，这里不做详述。
5、安装完成，激活windows
  • 网上有很多激活的操作方式，我个人推荐使用cmd激活，自行百度有教程。
-------------本文结束 感谢阅读-------------
