关于hexo无法托管到coding平台问题的解决办法

发表于 2019-03-25 | 分类于 学习笔记 | | 阅读次数:
字数统计: 312 | 阅读时长 ≈ 1

1、原因

自2019年以来，由于coding公司与腾讯开发者平台合作，coding的免费托管项目已经由腾讯开发者平台代理，所以网上教程hexo推送到coding的代码已经不适用了

2、解决办法

1、在腾讯开发者平台上创建新的秘钥，此秘钥就是你安装hexo的机器的ssh秘钥，的由于网络上的创建秘钥教程很多，所以这里不再重复。

2、登录腾讯开发者平台，创建自己的项目，项目名称最好是：你的用户名.coding.me，这样的话给你分配的域名会是你的项目名称，你的用户名如图所示。

3、单击自己建立的项目，右下角找到ssh推送网址，点击复制
